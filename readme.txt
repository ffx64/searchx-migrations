Database from xlog

----------------------------------------------------------------------
                           Database Files
----------------------------------------------------------------------

'*.sqlite' Files: These files are used in the test version with
SQLite. They are useful for quick development or testing environments.
You will find these files in the repository for easier local work.

'*.db' Files: These files are used for PostgreSQL. They contain the
database structure for production. When the system is set up for
PostgreSQL, the database files will be generated with the '.db'
extension.
